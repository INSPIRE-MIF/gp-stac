# Repository for discussing STAC as an INSPIRE Download service
This repository intends to collect evidence, facilitate discussions and possibly initiate a Good practice for the use of the SpatioTemporal Asset Catalog (STAC) as INSPIRE predefined-access download service.

Follow the instructions available [here](https://github.com/INSPIRE-MIF/gp-stac/tree/main/deployments) to report a deployment example. 

Use the [Issue Tracker](https://github.com/INSPIRE-MIF/gp-stac/issues/new) to start a discussion or report any issue related to the use of STAC.
